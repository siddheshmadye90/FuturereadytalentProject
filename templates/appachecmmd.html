*INFORMATION ABOUT APPACHE WEBSERVER*

A web server is computer software and underlying hardware that accepts requests via HTTP, the network protocol created to distribute web pages, or its secure variant HTTPS After configuration you can go on http://localhost:80 you can see the default web to change default web page with your own page you can do this by replacing the file present in /var/www/html with your html file also you can change the default port number by modifing apache config file you will find following lind binding Apache to port 80 change listen on 80 to listen on any port you want
