BASIC LINUX COMMANDS


*INFORMATION ABOUT LINUX*

LINUX is an operating system or a kernel distributed under an open-source license. Its functionality list is quite like UNIX. The kernel is a program at the heart of the Linux operating system that takes care of fundamental stuff, like letting hardware communicate with software.


  • Cal

    To open calender

  • Date

    Display date

  • Jobs

    Display running jobs / programs

  • fg<ProcessId>

    Start paused process

  • <command/process>&

    & will run command in background

  • which<FileName>

    Gives path of program/code file

  • gedit <command/program>

    Gedit is text editor it will open file with given name

  • mkdir <DirName>

    It will create directory with given name

  • rmdir <DirName>

    It will remove directory with given name

  • cd <FolderName>

    Change directory with given folder name or folder path

  • cat <fileName>

    Display content inside file

  • free -m

    shows memory(RAM) usage

  • history

    displays previously executed commands
